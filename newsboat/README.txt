# EXPORT CONFIGURATION #
########################

Newsboat is a CLI (terminal) RSS news reader.

First install it on your device. #[newsboat.org]

To export simply copy the <newsboat> directory to your home directory, and rename it <.newsboat>
To add your custom newsfeeds just modify the <urls> file.

You should have something like this to make it work:

/home/USERNAME/.newsboat/
├── config
└── urls
